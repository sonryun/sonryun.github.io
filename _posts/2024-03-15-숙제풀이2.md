            var display = document.querySelector('.display');
            if (display.value === '0') {
                display.value = '';
            }
            display.value += value;

<h5>숙제를 풀면서 배운 것</h5>
<b>eval 함수</b>
-eval 함수란 자바스크립트에 내장된 함수로, 식을 입력 받으면 그 식을 계산해서 결과를 알려주는 함수이다.

```html
eval(12+12*2) //결과 : 36, 연산자 우선순위도 고려한다고 함.
```
