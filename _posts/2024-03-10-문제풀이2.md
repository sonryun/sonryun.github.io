<h6>첫번째 문제</h6>
<p>
  -js에 연산자를 사용한 결과가 21이면 페이지에 Correct, My age is 21 이라는 창을 띄우고, 
그 이외의 결과값들은 Try again이라는 창을 띄운다.
</p>

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>quiz1</title>
</head>
<body>
    <button onclick="agecalc()">결과 보기</button>

    <script>
        function agecalc() {
            var x = 21, result;
            result = x;

            if (result === 21) {
                var str = 'Correct, My age is';
                alert (str + " " + result);
            }
            else {
                alert("Try again");
            }
        }
    </script>
</body>
</html>
```

<h6>두번째 문제</h6>
<p>- select css 를 이용하여 옵션을 누르면 alert 창이 나오는 문제 만들기</p>

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>quiz2</title>
</head>
<body>
    <select name="keroro" id="keroro" class="selectBasic" onchange="Rank()">
        <option value="케로로">케로로</option>
        <option value="도로로">도로로</option>
        <option value="타마마">타마마</option>
    </select>

    <script>
        function Rank() {
            var select = document.getElementById("keroro");
            var value = select.value;

            if(value === "케로로") {
                alert("중사입니다.");
            } else if(value === "도로로") {
                alert("병장입니다.");
            } else {
                alert("이등병입니다.");
            }
        }
    </script>
</body>
</html>

```

<h6>세번째 문제</h6>

```html

    </style>
```

<h6>네번째 문제</h6>
<p>
  -Q1.정답버튼을 눌렀을 때 무지개의 4번째 색인 Green이 alert창에 뜨게 하기. 옳지 않을 시에는 NaN -> Try again 이라는 창이 뜸
Q1-1.colorName 글자수의 짝/홀수를 감별하는 코드의 빈칸을 채워넣기.
</p>

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>quiz4</title>
</head>
<body>
    <h3>무지개의 4번째 색은 뭘까요?</h3>
    <button onclick="selectColor();">정답!</button>

    <script>
        function selectColor() {
            var color = 4;
            var colorName;

            switch (color) {
                case 1:
                    colorName = 'Red';
                    break;
                case 2:
                    colorName = 'Orange';
                    break;
                case 3:
                    colorName = 'Yellow';
                    break;
                case 4:
                    colorName = 'Green';
                    break;
                case 5:
                    colorName = 'Blue';
                    break;
                case 6:
                    colorName = 'Deepblue';
                    break;
                case 7:
                    colorName = 'Purple';
                    break;
                default:
                    colorName = NaN;
            }

            if (colorName === 'Green') {
                alert(colorName);
            } else {
                alert('Try again');
            }
        }
    </script>
</body>
</html>
```
