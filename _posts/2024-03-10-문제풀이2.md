<h6>첫번째 문제</h6>
<p>
  -js에 연산자를 사용한 결과가 21이면 페이지에 Correct, My age is 21 이라는 창을 띄우고, 
그 이외의 결과값들은 Try again이라는 창을 띄운다.
</p>

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>quiz1</title>
</head>
<body>
    <button onclick="agecalc()">결과 보기</button>

    <script>
        function agecalc() {
            var x = 21, result;
            result = x;

            if (result === 21) {
                var str = 'Correct, My age is';
                alert (str + " " + result);
            }
            else {
                alert("Try again");
            }
        }
    </script>
</body>
</html>
```

<h6>두번째 문제</h6>
<p>- select css 를 이용하여 옵션을 누르면 alert 창이 나오는 문제 만들기</p>

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>quiz2</title>
</head>
<body>
    <select name="keroro" id="keroro" class="selectBasic" onchange="Rank()">
        <option value="케로로">케로로</option>
        <option value="도로로">도로로</option>
        <option value="타마마">타마마</option>
    </select>

    <script>
        function Rank() {
            var select = document.getElementById("keroro");
            var value = select.value;

            if(value === "케로로") {
                alert("중사입니다.");
            } else if(value === "도로로") {
                alert("병장입니다.");
            } else {
                alert("이등병입니다.");
            }
        }
    </script>
</body>
</html>

```

<h6>세번째 문제</h6>
<p>-ㄱ과 ㄴ 중 ㄱ 고르면 o ,ㄴ 고르면 x나오게 하기</p>

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>quiz3</title>

    <style>
        body {
            font: bold;
            background-color: #f4f4f4;
            text-align: center;
            padding: 50px;
        }

        button {
            background-color: forestgreen;
            color: white;
            padding: 10px 20px;
            font-size: 16px;
            border: none;
            border-radius: 4px;
        }
    </style>
</head>
<body>
    <h1>다음 중 어떤 것이 ㄱ일까요?</h1>

    <button id="button1" onclick="question('ㄱ')">ㄱ</button>
    <button id="button2" onclick="question('ㄴ')">ㄴ</button>

    <script>
        function question(answer) {

            if(answer==="ㄱ") {
                alert("O");
            } else{
                alert("X");
            }
        }
    </script>
</body>
</html>
```
